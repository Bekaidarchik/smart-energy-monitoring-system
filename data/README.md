# Data

This folder stores voltage, current, power, and energy records using the shared
project schema:

```text
timestamp_s,voltage_v,current_a,power_w,energy_j,load_type,measurement_status,notes
```

Current status: simulation-first.

The included CSV files are generated by `analysis/energy_analysis.py`. They are
marked with `measurement_status = simulated` so the project does not claim real
hardware results before the Arduino/ESP32 prototype is tested.

When hardware is available, replace or add to `raw_measurements.csv` with
serial output captured from the INA219 logger, then rerun the analysis script.
The script accepts either the project schema or the Arduino serial schema:

```text
timestamp_ms,voltage_v,current_ma,power_mw
```

If the Arduino format is used, milliseconds are converted to seconds and
milliamps are converted to amperes during cleaning.
