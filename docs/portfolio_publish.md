# Portfolio Publishing Guide

## GitHub Repository

Repository name:

```text
smart-energy-monitoring-system
```

GitHub description:

```text
Low-cost Arduino/ESP32 energy monitoring prototype for voltage, current, power logging, and Python-based energy-use analysis.
```

Recommended portfolio position:

```text
Pin this as the second pinned GitHub project after EcoStep.
```

## Upload Steps

1. Create a new GitHub repository named `smart-energy-monitoring-system`.
2. Set the description to the text above.
3. Upload the contents of this project folder, not the parent `outputs` folder.
4. Confirm that the README renders the four SVG figures.
5. Pin the repository on the GitHub profile after EcoStep.

## Current Publishing Constraint

The local project package is complete, but this environment does not currently
provide Git, GitHub CLI, or a GitHub connector method for creating a brand-new
repository. The included ZIP file can be uploaded through GitHub's web
interface, or the folder can be committed from a machine with Git installed.
